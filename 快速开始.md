## 使用方式

### 方式 1：连接局域网 MySQL（推荐）

1. 复制 `config/db_config.example.json` → `config/db_config.json`，填写内网数据库信息。
2. 测试连接：`python database\db_connection.py`
3. 按需运行导入：`python scripts\dataImport\run_batch.py`

路径与日期默认读取 `config/path_config.py`（共享盘 `\\Betohow\...`）。

### 方式 2：本地 Docker MySQL

```powershell
.\scripts\docker-up.ps1
```

首次启动会自动执行 `database/schema.sql`。连接端口默认 **3307**（见 `docker-compose.yml`）。

### 方式 3：v2 脚本（.env 配置）

在根目录配置 `.env`（参考 `.env.example`），然后：

```powershell
python scripts\v2\orders\run_import.py
python scripts\v2\profit\run_order_sku_profit.py
```

详见 [scripts/v2/README.md](scripts/v2/README.md)。

## 常见问题

### 数据库连接失败

1. 检查 `config/db_config.json` 或 `.env`
2. 若用 Docker：`docker compose ps` 确认容器健康
3. 确认防火墙与内网可达

### 找不到 Excel 文件

1. 确认共享盘 `\\Betohow\...` 已挂载
2. 检查 `config/path_config.py` 中 `DATE_PATH` 是否与 ERP 落盘日期一致
