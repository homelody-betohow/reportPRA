# 快速开始

## 1. 安装依赖

```powershell
cd <项目根目录>
pip install -r requirements.txt
```

## 2. 配置数据库

复制 `config/db_config.example.json` 为 `config/db_config.json`，填写 MySQL 连接信息：

| 项 | 说明 |
|----|------|
| host | MySQL 服务器 IP |
| port | 端口 |
| user / password | 账号密码 |
| database | 库名（如 `rpa-report`） |

测试连接：

```powershell
python database\db_connection.py
```

## 3. 配置共享盘路径（可选）

编辑 `config/path_config.py`：

- `MODE_RUN`：`日报` / 月报
- `DATE_PATH`：默认日报为 3 天前日期
- `ERP_ORDER_STA_PATH` 等：共享盘根路径

本机须能访问 `\\Betohow\...` 网络路径。

## 4. 批量导入订单数据

```powershell
python scripts\dataImport\run_batch.py
```

常用参数：

```powershell
python scripts\dataImport\run_batch.py --date 2026-06-09
python scripts\dataImport\run_batch.py --import-batch 20260616_120000
python scripts\dataImport\run_batch.py --mode 每月
```

## 5. 计算利润

```powershell
python scripts\archive\run_batch.py
```

批次号默认从 `scripts/dataImport/run_batch.lock` 读取；也可 `--batch` 手动指定。

## 6. Docker 方式（可选）

```powershell
.\scripts\docker-up.ps1
```

或：

```powershell
docker compose up -d
```

- MySQL 端口、账号见 `.env`（默认 `3306` / `rpa-bth`）
- 首次建表：按 `docs/database/*.sql` 在库中执行 DDL
- 详见 [Docker部署.md](Docker部署.md)、[MySQL-连接说明.md](MySQL-连接说明.md)

## 快速测试

```powershell
# 单步导入示例
python scripts\dataImport\order_shipped.py --date 2026-05-26

# 配置向导（可选）
python config\run_config.py
```

## 常见问题

**数据库连接失败**：检查 `config/db_config.json`；Docker 环境执行 `docker compose ps`。

**找不到 Excel**：确认共享盘已挂载，`path_config.py` 中 `DATE_PATH` 与 ERP 落盘日期一致。

**利润批次号为空**：先执行 `dataImport/run_batch.py`，或 `archive/run_batch.py --batch <批次号>`。
